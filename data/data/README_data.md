# Fuente de datos
Este proyecto utiliza un dataset público.

## Dataset
- Fuente: https://www.kaggle.com/datasets/somesh140/segmentation
- Archivo esperado: `customer_data.csv`

## Cómo usar
1. Descargar el dataset desde el link
2. Guardarlo dentro de la carpeta `/data/`
3. Ejecutar el notebook en `/notebooks/`
